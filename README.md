# 目标分析
确定目标，然后对目标进行分析（选择策略），包含三个分析方向
在页面中通过右键审查元素的方式来获取一下三个元素的信息
   URL格式：限定抓取网页的范围
   数据格式：根据需要抓取的信息，选择抓取的标签
   网页编码：在代码解析器中指明编码格式
